# Сервис назначения ревьюеров для Pull Request’ов
Данный микросервис назначает ревьюеров на PR из команды автора, позволяет выполнять переназначение ревьюверов и получать список PR’ов, назначенных конкретному пользователю, 
а также управлять командами и активностью пользователей.
## Запуск сервиса
1. Клонируйте репозиторий
```bash
git clone https://github.com/ayanosky/pr-reviewer-service.git
cd pr-reviewer-service
```
2. Сборка образа приложения и последующий запуск контейнера
```bash
docker-compose build
docker-compose up
```
3. Так же можно использовать Makefile для сборки и запуска сервиса
```bash
make build
make run
```
